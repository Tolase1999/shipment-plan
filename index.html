How to Set Up Google Sheets Connection

Step 1: Create Your Google Sheet

Create a new Google Sheet with this exact column structure in the first sheet (Sheet1):

  • Column A: Office Name
  • Column B: Address
  • Column C: Phone
  • Column D: Contact Person

Row 1 should be headers. Data starts from Row 2.

Step 2: Make Sheet Public

Click Share → Change to Anyone with the link can view.

Step 3: Get Sheet ID

Copy the long string from your sheet URL:
https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit

Step 4: Get API Key

1. Go to Google Cloud Console
2. Create a new project
3. Enable Google Sheets API
4. Go to Credentials → Create API Key
5. Copy the API Key

Step 5: Connect

Paste both values above and click Load Offices.
